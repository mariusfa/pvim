# PVIM
Personal neovim config

## Git clone
```bash
git clone git@github.com:mariusfa/pvim.git ~/.config/nvim
```

## Clean nvim state
```bash
rm -rf ~/.local/share/nvim ~/.local/state/nvim ~/.cache/nvim
```
